GeoExplorerTourism es una aplicación web orientada al turismo que permite a los usuarios explorar, 
visualizar y gestionar información sobre distintos lugares turísticos de manera interactiva.

El sistema está diseñado bajo una arquitectura full stack, integrando un frontend desarrollado con Angular y un backend basado en Node.js con Express,
lo que permite una comunicación eficiente entre la interfaz de usuario y la base de datos.
